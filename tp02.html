1- ¿que es una URL?

1- una URL es una cadena de texto que especifica dónde puede encontrarse un recurso, también conocido como dirección WEB o enlace.


2- ¿que es url absoluta?

2- una URL absoluta es una dirección web completa que incluye el protocolo, el nombre de dominio y la ruta completa del archivo o recurso

3-¿qué es una url relativa?

Las URL relativas son las que describen la ruta a un archivo relativo al actual documento

4-¿cuál es la diferencia entre una url relativa y una url absoluta?

4- La diferencia entre la URL absoluta y relativa es que la relativa no requiere el protocolo o nombre de dominio, solo el nombre de ruta y la absoluta si lo requiere
